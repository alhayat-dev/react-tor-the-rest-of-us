Step 1: Install prettier vscode extension and place the following options in the settings.json

{
"files.associations": {
"\*.js": "javascriptreact"
},
"editor.defaultFormatter": "esbenp.prettier-vscode",
"editor.formatOnSave": true,
"prettier.semi": false,
"prettier.printWidth": 9999,
"editor.tabSize": 2
}
